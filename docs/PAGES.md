# GitHub Pages

- source page: `docs/index.html`
- deploy workflow: `.github/workflows/pages.yml`
- target URL: `https://hwkim3330.github.io/openai/`

## Enable (once)
1. GitHub repo > Settings > Pages
2. Build and deployment: `GitHub Actions`
3. Push to `master`/`main`

## Notes
- Pages app is fully static and keyless.
- For secure/private automation, use Node runtime + Telegram allowlist.
