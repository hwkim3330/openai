# Commands

## Built-in
- `help`
  - show command list
- `browse: <url>`
  - fetch and summarize page text via Playwright
- `weather: <city|lat,lon>`
  - current weather via Open-Meteo
- `traffic: <from> -> <to>`
  - route ETA/distance via OSM+OSRM
- `stock: <sym1,sym2,...>`
  - daily quote via Stooq (`aapl.us` style recommended)

## Telegram examples
If prefix is `!`:
- `!help`
- `!weather: Seoul`
- `!traffic: Seoul Station -> Incheon Airport`
